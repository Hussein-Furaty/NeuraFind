# NeuraFind

NeuraFind is a Windows desktop application for semantic document search.

Instead of relying on filenames or exact keyword matching, the application indexes the textual content of local documents and retrieves results based on semantic similarity.

## Goals

The project aims to provide:

- Local document indexing
- Semantic document retrieval
- Offline search after indexing
- A modular and maintainable architecture

## Current Scope

The first development milestone focuses on support for:

- PDF
- Microsoft Word (.docx)
- Microsoft Excel (.xlsx)
- Microsoft PowerPoint (.pptx)
